Le Chemin Vers L'édification

Les reliques passées des sélecteurs spécifique aux navigateurs, des DOMs incompatibles, et du manque de support des CSS encombrent un long chemin sombre et morne.

Aujourd'hui, nous devons nous clarifier l'esprit et nous débarrassez des pratiques passées. La révélation de la véritable nature du Web est maintenant possible, grâce aux efforts infatigables des gens du W3C, du WaSP et des créateurs des principaux navigateurs.

Le Jardin Zen css vous invite à vous relaxer et à méditer sur les leçons importantes des maîtres. Commencez à voir clairement. Apprenez à utiliser ces techniques (bientôt consacrées par l'usage) de manière neuve et revigorante. Ne faites qu'Un avec le Web.

Alors, De Quoi S'agit–il?

Il y a clairement un besoin pour les graphistes de prendre les CSS au sérieux. Le Jardin Zen vise à exciter, inspirer, et encourager la participation. Pour commencer, voyez quelques concepts choisis dans la liste. Cliquez sur n'importe lequel pour le charger sur cette page. Le code HTML demeure le même, et seule la feuille de style extérieure change. Oui, vraiment.

Les CSS permettent un contrôle complet et total du style d'un document hypertexte. La seule manière de vraiment démontrer cela d'une manière qui excite les gens est de démontrer ce qui peut vraiment être, une fois que les rennes ont été placées dans les mains de ceux capables de créer la beauté basée sur la forme. Jusqu'à maintenant, les exemples de trouvailles et montages intéressants ont étés fournis par des programmeurs et des structuralistes. Les concepteurs ont encore à faire leurs preuves. Cela doit changer.

Participation

Graphistes seulement s'il vous plaît. Vous aller devoir modifier cette page, donc une compétence solide en CSS est nécessaire, mais les documents exemples sont suffisamment commentés pour que les novices en CSS puisent les utiliser comme points de départ. S'il vous plaît, consultez le Guide de Ressources CSS pour des tutoriaux avancés et de l'information pour travailler avec les CSS.

Vous pouvez modifier la feuille de style de n'importe quelle manière, mais pas le code HTML. Ceci peut sembler décourageant au début si vous n'avez jamais travaillé de cette manière, mais suivez les liens pour en apprendre plus, et utilisez les exemples pour vous guider.

Téléchargez fichiers d'exemple html et css pour travailler avec une copie locale. Une fois que vous avez fini votre chef d'oeuvre (ne soumettez pas de travaux incomplets), téléchargez votre document . css sur un serveur web sous votre contrôle. Envoyez-nous un lien vers votre document, et si nous le sélectionnons, nous relèverons les images nécessaires.

Mise à jour: je cherche des traducteurs. Si vous parlez l'anglais couramment ainsi qu'une autre langue, vous auriez ainsi la possibilité de participer d'une autre manière au projet. S'il vous plaît, contactez-moi pour plus de détails. Traduction par Nicolas Steenhout et Marie Alhomme.

Bénéfices

Pourquoi participer? Pour être reconnu, pour l'inspiration, et pour créer une ressource que nous pouvons tous utiliser quand nous voulons promouvoir le cas de la conception web à base de CSS.

Critères

Nous aimerions voir autant de CSS1 que possible. CSS2 devrait être limité aux élements bien supportés. Le Jardin Zen css est basé sur une approche pratique et fonctionelle des CSS, et non sur les derniers procédés lisibles seulement pour 2% du publique. Le seul vrai critère est que votre CSS puisse être validé.

Malheureusement, conceptualiser de cette manière rehausse les défauts des différentes implémentations CSS. Differents navigateurs affichent de manière différente le même code, des fois même avec une CSS complétement valide, et ça devient enrageant quand une solution pour un navigateur équivaut à un nouveau problème dans un autre. Consulter la page de Ressources pour de l'information au sujet de quelques solutions disponibles. La conformité des navigateurs appartient encore au domaine du rêve, et nous ne nous attendons pas à ce que vous obteniez avec un code parfait sur toutes les plates-formes. Mais testez avec autant de navigateurs et plates-formes que possible. Si votre concept ne fonctionne pas au moins avec IE5+/Win et Mozilla (utilisé par 90% de la population), les chances sont bonnes que nous ne l'acceptions pas.

Nous demandons que vous soumettiez des graphismes originaux. Veuillez respecter les lois du Copyright. S'il vous plaît, veuillez garder le matériel choquant ou inacceptable au minimum; la nudité légère et justifiée est acceptable, la pornographie sera rejetée.

Ceci est un exercice d'apprentissage. Vous gardez le Copyright complet de vos graphiques, mais nous vous demandons de céder votre fichier CSS sous une licence Creative Commons identique à celle sur ce site (en anglais) afin que d'autres personnes puissent en apprendre.
